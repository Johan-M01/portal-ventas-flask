# Portal de Ventas (Flask)

Este es un sistema simple para registrar ventas con notas y monto, desarrollado en Python usando Flask + SQLite.

## Cómo ejecutar

1. Asegúrate de tener Python y Flask instalados:
   pip install flask flask_sqlalchemy

2. Ejecuta la app:
   python app.py

3. Abre tu navegador:
   http://127.0.0.1:5000/
